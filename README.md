# React + Vite + Tailwind + Daisyui

This template provides a minimal setup to get React working in Vite with
Tailwind CSS .

## Installation

For Development...

```sh
git clone https://github.com/mhmamun86/react-blank-setup.git
cd react-blank-setup
npm install
npm run dev
```

For production...

```sh
npm run build
```
